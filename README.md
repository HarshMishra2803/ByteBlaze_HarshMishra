## 🚀 Project Overview

**LegalDoc AI** is a full-stack web application designed to simplify legal document creation, analysis, and understanding using Artificial Intelligence.

Legal processes are often complex, expensive, and difficult to understand for non-legal users. Our platform solves this problem by providing an easy-to-use AI-powered solution for generating, analyzing, and managing legal documents.

---

## 🎯 Key Features

### 📄 Document Generator

* Generate legal documents instantly:

  * NDA (Non-Disclosure Agreement)
  * Rental Agreement
  * Employment Contract
* Structured and professional output

---

### 📊 Upload & Analyze

* Upload documents (PDF / TXT)
* AI-powered features:

  * Clause explanation
  * Compliance & risk analysis

* Chat-based legal assistant
* Answer queries in simple language
* Explain complex legal concepts

---

### 📥 PDF Export

* Download clean formatted documents
* Supports English & Hindi

---

### 🛠️ Admin Panel (Advanced)

* Create & manage templates
* Dynamic form generation
* Scalable system

---

## 🧠 How It Works

1. User selects template or uploads document
2. AI processes input using LLM APIs
3. Structured output is generated
4. Results are displayed in a clean UI
5. User can download or analyze further
